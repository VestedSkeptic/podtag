#podtag
podtag is a simple tool to set mp3 tags in podcasts. 

##Usage
usage: podtag.py [-h] [--i [input_file]]

optional arguments:
  -h, --help        show this help message and exit
  --i [input_file]  text file containing items to back up, default is
                    'podcast.txt'
